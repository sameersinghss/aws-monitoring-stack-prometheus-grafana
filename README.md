# AWS Monitoring Stack using Prometheus, Node Exporter and Grafana

## Overview

This project demonstrates the deployment of a complete monitoring and observability stack on AWS EC2 using Prometheus, Node Exporter, and Grafana.

## Technologies Used

* AWS EC2
* Ubuntu Linux
* Prometheus
* Node Exporter
* Grafana
* Systemd
* Linux Networking

## Features

* CPU Monitoring
* Memory Monitoring
* Disk Monitoring
* Network Monitoring
* Uptime Monitoring
* Real-time Dashboard Visualization

## Architecture

EC2 Ubuntu Server
↓
Node Exporter
↓
Prometheus
↓
Grafana

## Challenges Solved

* SSH Timeout Issue
* Prometheus Service Startup Failure
* Linux Permission Errors
* Disk Quota Exceeded Error
* Prometheus Target Configuration

## Results

Successfully deployed a monitoring solution with Prometheus and Grafana on AWS EC2 and visualized infrastructure metrics in real time.
